# Learning Online at TC

A student-facing prototype web page for Teachers College, Columbia University's Digital Futures Institute (DFI). Built as a single-file static HTML prototype with inline CSS and vanilla JavaScript.

## Viewing

Open `index.html` in any modern browser, or visit the hosted version on GitHub Pages (if enabled).

## Structure

- `index.html` — the full prototype
- `assets/` — images (hero, step illustrations, logos, background patterns)

## Topics covered

- Welcome and orientation video
- How TC's online learning works (asynchronous + synchronous, tools, accessibility)
- Getting set up (accounts, equipment, readiness check)
- Succeeding as an online student (study skills, communication, wellbeing)
- AI at TC (expectations, responsible use, institutional tools, AI literacy)
- Support and contact directory

## Authoring

Created in collaboration with DFI at Teachers College, Columbia University.
